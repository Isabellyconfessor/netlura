# 🍿 Netlura - Imersão Front-End na Era da IA

Bem-vindo ao repositório do **Netlura**, um projeto com interface inspirada na Netflix, desenvolvido como desafio da **Imersão Front-End na Era da IA** promovida pela Alura.

🔗 **[Acesse o projeto rodando online aqui!](https://isabellyconfessor.github.io/netlura/)**

## 💻 Sobre o Projeto

A ideia desse projeto foi ver na prática como a Inteligência Artificial pode facilitar a nossa vida na hora de programar. A proposta da Imersão da Alura era criar uma página com a interface da Netflix, usando a IA como nossa parceira de código.

### 🛠️ Como foi construído:
* **Página de Catálogo:** Utilizada uma estrutura pronta e disponibilizada previamente no GitHub da imersão.
* **Construção e Estilização:** Todo o restante da interface e da estruturação do código foi desenvolvido com o auxílio da Inteligência Artificial nativa do VS Code. A IA atuou como um "pair programming", ajudando a gerar componentes e acelerar a codificação da página.

## 🚀 Tecnologias e Ferramentas Utilizadas

* **HTML5 & CSS3**
* **JavaScript**
* **VS Code + IA Integrada** (Geração de código e assistência)
* **GitHub Pages** (Hospedagem)

## ✨ Aprendizados

Este projeto foi fundamental para entender, na prática, como as inteligências artificiais podem otimizar o tempo de desenvolvimento e atuar como assistentes poderosas na estruturação do código. O grande desafio e aprendizado foi saber como pedir ("promptar") as estruturas corretas para a IA e ajustar o código gerado para que ele funcionasse perfeitamente no navegador.
